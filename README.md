# 多店铺手机实体店铺ERP系统

一套完整的手机零售行业企业资源管理解决方案，支持多店铺管理、库存同步、销售管理、财务管理等核心功能。

## 技术栈

- **前端框架**: Uniapp (Vue3 + TypeScript)
- **UI组件库**: uView Plus
- **状态管理**: Pinia
- **后端服务**: uniCloud (Serverless)
- **数据库**: uniCloud 云数据库 (MongoDB兼容)
- **构建工具**: Vite

### 4. 销售管理

### 6. 报表分析
- 销售报表
- 库存报表
- 财务报表
- 员工业绩报表

## 用户角色

| 角色 | 说明 | 核心权限 |
|-----|------|---------|
| 超级管理员 | 系统最高权限 | 全系统管理、店铺管理、用户管理 |
| 店铺管理员 | 单店铺管理者 | 店铺管理、员工管理、商品库存管理 |
| 收银员 | 门店收银人员 | 收银操作、会员查询、销售记录查看 |
| 仓库管理员 | 库存管理人员 | 库存管理、入库出库、盘点操作 |

## 快速开始

### 环境要求
- HBuilderX 3.6+ 或 VS Code
- Node.js 16+
- uniCloud 账号（阿里云或腾讯云）

### 项目初始化
1. 使用 HBuilderX 导入项目
2. 配置 uniCloud 云服务空间
3. 上传云函数和数据库Schema
4. 运行项目到小程序/App/H5

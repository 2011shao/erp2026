# 多门店 ERP 系统功能说明

## 系统概述
多门店 ERP 系统是一个专为线下手机与 3C 产品门店设计的企业资源管理系统，旨在帮助企业实现对多门店的统一管理，提高运营效率和管理水平。

## 主要功能模块

### 1. 首页
- 系统概览：展示总店铺数、总商品数、今日销售额、库存总量等关键指标
- 数据可视化：通过卡片形式展示核心业务数据

### 2. 店铺管理
- 店铺信息管理：添加、编辑、删除店铺信息
- 店铺列表：展示所有店铺的详细信息
- 店铺信息包括：店铺名称、地址、电话、负责人

### 3. 商品管理
- 商品信息管理：添加、编辑、删除商品信息
- 商品列表：展示所有商品的详细信息
- 商品信息包括：商品名称、分类、品牌、型号、价格、成本价、库存、所属店铺

### 4. 库存管理
- 库存预警：展示库存不足的商品
- 库存列表：展示所有商品的库存情况
- 库存变动记录：记录所有库存的入库和出库操作
- 库存调整：支持手动调整库存数量

### 5. 销售管理
- 订单管理：添加、查看销售订单
- 订单列表：展示所有销售订单的详细信息
- 订单状态管理：支持订单状态的更新（待处理、已完成、已取消）
- 订单详情：查看订单的商品明细

### 6. 财务管理
- 财务记录管理：添加、删除财务记录
- 财务记录列表：展示所有财务记录的详细信息
- 收支统计：展示总收入、总支出、净利润等财务指标
- 财务记录包括：店铺、金额、类型（收入/支出）、分类、描述、创建时间

### 7. 报表分析
- 销售趋势：通过折线图展示半年销售趋势
- 库存状况：通过柱状图展示商品库存情况
- 财务分析：通过柱状图展示财务收支情况
- 店铺销售对比：通过饼图展示各店铺销售占比
- 统计卡片：展示总销售额、总利润、总订单数、商品种类等指标

### 8. 用户管理
- 用户信息管理：添加、编辑、删除用户信息
- 用户列表：展示所有用户的详细信息
- 用户角色管理：支持管理员、店长、员工三种角色
- 用户信息包括：用户名、角色、所属店铺、创建时间

## 技术特点
- 响应式设计：适配不同屏幕尺寸
- 现代化界面：使用 Ant Design 组件库
- 模块化架构：清晰的功能模块划分
- 实时数据：即时更新的数据展示
- 友好的用户体验：直观的操作界面

## 系统要求
- 前端：React 18 + TypeScript + Ant Design 5.x
- 后端：Express.js + TypeScript
- 数据库：SQLite（开发环境）
- 构建工具：Vite 5.x
- 状态管理：Zustand
- 路由：React Router v6
- 图表：Chart.js

## 系统价值
- 提高管理效率：统一管理多门店的业务数据
- 优化决策：基于数据报表进行科学决策
- 降低运营成本：减少人工操作和错误
- 提升服务质量：及时响应客户需求
- 增强企业竞争力：规范化管理流程
